# facemask-detection

this project is about to detect face-mask

# Data set
https://github.com/chandrikadeb7/Face-Mask-Detection

# Framework / technic
- tensorflow=1.13.1 
- keras=2.2.4
- facenet 
- mobilenetV2

# How to user
1. download dataset
2. download model
3. train network with
    > $ python mask_train.py
4. detect face mask with
    > $ python face.py
